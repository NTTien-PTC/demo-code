T2: vai-tay sau 
- Tập vai
+ đẩy vay với thanh đòn 2 set khởi động 2 set chính, set cuối thêm 1 set bay vai với bánh tạ(20c)
+ đẩy vai ngồi ghê tạ đơn 12c super set với bay vai 15c 4set
+ đẩy vai 1 tay, đứng với tạ đơn 12-15c superset với bay vai về phía trước 12c 3-4 set 
+ giựt vai(vai sau) 12-15c superset với plank 1 phút 3set
- Tập tay sau(4 set lần lượt các bài)
+  đứng đẩy tay sau bằng thanh cong 10-12c
+  đứng đẩy tay sau bằng tạ đơn 10-12c
+  nằm đẩy thanh đòn(ngực giữu) khép cùi trỏ 10-12c
T3: ngực-bụng
- Tập ngực
+ ngực trên thanh đòn 4set 2 set khỏi động 12,12c 2 set 8c,8c
+ ngực dưới thanh đòn 12-10-8 
+ dips 3 set hết sức 
+ kéo cáp từ trên xuống(5c) từ xuống lên (5c) 2 set, dây cáp ở trên
+ kéo cáp từ dưới lên(dây cáp ở dưới) 14c superset với hít đất 15-20c 3set
- Tập bụng 
+ đá bụng với xà 3set hết sức(lúc đá lên hơi kéo lưng để cố định)
+ đá bụng dưới tập bụng dưới  12c superset tập bụng với ghế nghiêng 12c 3set
+ đá bụng với xà liên sườn hết sức
T4: lưng - cẳng tay 
- Tập lưng
+ last pull down 6set 2 set đầu khởi động 20c 4 set sau 12-12-10-8 	
+ deadlift 4set 8c
+ kéo với máy 4set 2tay 10c 1tay 10c
+ kéo cáp  ngồi(kéo ngang thanh tam giác) 12c superset với kéo dây thừng đứng 10c 4set
+ kéo cáp ngồi(kéo từ trền xuống thanh tam giác) 10c superset last pull down sau đầu 10c
- tập cẳng tay
+ cuốn cẳng tay bằng thanh cong sau người 15c superset với cuốn tạ tay với tạ đơn theo chiều dọc 12c
- Tập cẳng tay 
T5: full chân
- squat 12c superset với đá chân với máy 12c 3set
- sumo deadlift(mở góc 45 độ, tay gần vai, ưởn ngực lên, cao mông ) 8c  superset cuộn chân với máy 15c
- đá chân với máy 2 chân 12c superset 1 chân 10c 3set
- banh chân từ ngoài vào trong 12c,banh chân từ trong ra ngoài 12c, nhún chân mỗi chân 15c
T6: full tay
- tay trước với thanh cong 20c 2 set
- tay sau đẩy với thanh đòn(đẩy ngực giữa) 20c 2set
- đẩy tay sau với tạ đơn(như kéo xô) 12c-10c 3set
- đẩy tay sau, sau đầu với bánh tạ 12c-10c 3set
- tay trước với tạ đơn chiều dọc 12c 4 set
- tay tược cuốn ngồi tạ ngang hết sức
